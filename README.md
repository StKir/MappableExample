# Example of using react-native-mappable

- Recommended settings

```
yarn: ~3
node: >=18
cocoapods: 1.15.2
```

- Install dependencies

```sh
yarn install
cd ios
pod install
```

- Paste the key for the maps into the app/examples/api_keys.ts file

```ts
export const MAP_KEY = 'MAP_KEY';
```

- You can run the application
